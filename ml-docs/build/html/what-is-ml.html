What is Machine Learning?

Machine learning (ML) is a branch of artificial intelligence that focuses on building systems that learn from data.

Key Concepts

  • Supervised Learning: Learning with labeled data.

  • Unsupervised Learning: Discovering patterns from unlabeled data.

  • Reinforcement Learning: Learning to make sequences of decisions.
